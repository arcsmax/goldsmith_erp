**Owner:** Max Kull. Escalation: Meister Thomas for workshop reality.

Open this file before putting a date on the calendar. Every constraint
below is a hard "no" — none are nice-to-haves.

## Weekday rule

**Ship only on Tuesday or Wednesday.**

Never on:

- **Friday** — goldsmith cannot raise a bug before the weekend.
- **Monday** — weekend rust + tool-setup overhead; users won't tolerate
  friction on a Monday, adoption cap triggers immediately.
- **Saturday / Sunday** — workshop closed.
- **Thursday** — borderline; reserved for fallback only.

**Target:** `2026-06-16 Tue` (Ship-Day Calendar).
**Fallback 1:** `2026-06-17 Wed` (Ship-Day Calendar).
**Fallback 2:** `2026-06-23 Tue` (slip by one week, see slip rule below).
**Rejected:** Tuesday slips that land on a public holiday — use the
next Tue / Wed that isn't blacked-out.

## Slip rule

If ship slips, do not slip by one day — slip by one full week to the
same weekday.

Concrete:

- If `2026-06-16 Tue` blocked → try `2026-06-17 Wed`.
- If both `2026-06-16 Tue` AND `2026-06-17 Wed` blocked → slip to
  `2026-06-23 Tue`. Kai Richter rejected the Thu alternative.

Conditions that trigger the slip from W9 to W10 (per DECISIONS-2026-04-16
and Ship-Day Calendar W9 risk watch):

- **Meister Thomas's V5 commitment not in writing by 2026-05-01 EOD.**
  Slip directly to `2026-06-23 Tue`.
- **DPIA returns red at DPO review on 2026-06-05.** Slip directly to
  `2026-06-23 Tue`.
- **Any dogfood blocker from Lena §5 unresolved by 2026-06-15.** See
  `V1.1-dogfood-protocol.md` for the blocker list.

A Wednesday-only slip is permitted only when the blocker is a single-
day operational delay (e.g. scanner hardware RMA that resolves
overnight). Design-level blockers always require the one-week slip.

## Pre-ship 48h rule (Anna A14.5 / V7)

Art. 13 BDSG §26 staff notification on `client_tap_at` logging must
reach the workshop at least **48 hours before** the ship commit lands
in production. On ship-day Tuesday 2026-06-16, this means the
notification must be in the workshop inbox by **Sunday 2026-06-14
EOD**.

This is a legal gate, not an operational one. Missing it is a stop-ship.
